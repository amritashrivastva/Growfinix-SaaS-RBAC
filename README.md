# Growfinix Internship - Task 2

## SaaS Multi-Tenant RBAC System

This project was developed as part of the **Growfinix Internship - Task 2**.

## Tech Stack

- Next.js
- Express.js
- Prisma ORM
- PostgreSQL (Neon)
- JWT Authentication
- bcrypt.js

## Features

- User Signup
- User Login
- JWT Authentication
- Role Based Access Control (RBAC)
- Super Admin Dashboard
- Manager Dashboard
- User Dashboard
- Users List
- Change User Role
- Delete User
- Logout
- Multi-Tenant Architecture

## Roles

- SUPER_ADMIN
- MANAGER
- USER
